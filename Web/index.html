sendAjax

此函数用来发送Ajax请求

语法

                    var modelData = {...};
                    var url = '/web/project/listpost'
                    var f1 = function(res) {
                        console.log(res);
                    };
                    sendAjax(modelData, url, f1);

更多细节

modelData 要传的数据
url 请求的网址
fn1 res.code === 1时执行
fn2 res.code === 2000时执行
fn3 res.code === 2003时执行
fn3 res.code返回了除上面以外的值时执行
